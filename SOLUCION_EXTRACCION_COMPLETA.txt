🚀 INICIO RÁPIDO (3 PASOS):

1. CONFIGURAR API KEY
   --------------------
   a) Ir a: https://console.cloud.google.com/
   b) Crear proyecto y habilitar "Places API"
   c) Crear API key
   d) Copiar .env.example a .env
   e) Pegar API key en .env

2. PROBAR API KEY
   ---------------
   cd backend
   python test_google_places.py
   
   Debe mostrar: "✅ TU API KEY FUNCIONA CORRECTAMENTE"

3. EXTRAER SUPERMERCADOS
   ----------------------
   python extract_google_places.py
   python populate_from_google_places.py
   
   Resultado: 600-800 supermercados reales en tu BD
